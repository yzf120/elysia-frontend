## 🐛 常见问题排查

### 问题1：验证码发送失败

**错误信息**：`短信模板未配置` 或 `验证码发送失败`

**原因**：后端短信服务未配置

**解决方案**：
- 使用密码登录方式测试
- 或联系后端配置短信服务

---

### 问题2：登录后跳转404

**错误信息**：页面显示 404 Not Found

**原因**：对应的后台页面尚未创建

**解决方案**：
- 这是正常现象，说明登录功能正常
- 可以通过检查 localStorage 确认登录成功
- 后续创建对应的后台页面即可

---

### 问题3：接口请求失败

**错误信息**：`Network Error` 或 `timeout`

**排查步骤**：
1. 检查后端服务是否启动
   ```bash
   curl http://localhost:8001/api
   ```

2. 检查浏览器控制台 Network 面板
   - 查看请求状态码
   - 查看请求和响应数据

3. 检查 API 地址配置
   - 打开 `src/api/index.js`
   - 确认 `baseURL` 是否正确

---

### 问题4：样式显示异常

**现象**：页面样式混乱或不显示

**排查步骤**：
1. 检查 CSS 文件是否正确加载
   - 打开浏览器开发者工具
   - 查看 Network 面板中的 CSS 请求

2. 检查 HTML 中的 CSS 引用路径
   ```html
   <link rel="stylesheet" href="/src/styles/global.css">
   ```

3. 清除浏览器缓存并刷新
