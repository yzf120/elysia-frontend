[//]: # (## 🔐 认证机制)

[//]: # ()
[//]: # (1. **登录成功后**：)

[//]: # (   - Token 保存到 `localStorage`)

[//]: # (   - 用户信息保存到 `localStorage`)

[//]: # (   - 用户类型保存到 `localStorage`)

[//]: # ()
[//]: # (2. **请求拦截**：)

[//]: # (   - 自动在请求头添加 `Authorization: Bearer <token>`)

[//]: # ()
[//]: # (3. **响应拦截**：)

[//]: # (   - 401 错误自动清除登录信息并跳转首页)

[//]: # (   - 统一错误提示处理)

[//]: # ()
[//]: # (4. **路由守卫**：)

[//]: # (   - 保护需要登录的页面)

[//]: # (   - 验证用户类型匹配)
